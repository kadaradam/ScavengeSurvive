# Scavenge and Survive Python Package (pyss)

A Python helper package with interfaces to Scavenge and Survive data files.

Also includes a set of command-line scripts and utilities for generating maps.


# Usage

`examples.py` will demonstrate the usage and ccapabilities of this package.
The package comes with a `setup.py` script but this isn't necessary.
Scripts in the root directory can be executed with arguments:
- `render_map.py` renders a set of .jpeg maps to `data/`.
- `generate_trees.py` generates .tpl files for tree placement.
